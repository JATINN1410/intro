# Intro Portfolio

This is a small personal portfolio site (static HTML/CSS/JS).

What's included
- `index.html`, `index.css`, `index.js` — main portfolio
- `images/` — assets
- `music/`, `resume_builder/` — project demos linked from the portfolio

Recent improvements
- Fixed invalid HTML tags and removed extra commented code
- Polished button styles, improved readability and hover states
- Restyled the comment box and added friendly form behavior
- Added a back-to-top button and subtle background accent

How to preview
1. Open `index.html` in your browser (double-click) or serve with a simple HTTP server.

	- Python 3:
		```bash
		python -m http.server 8000
		```
	- Then open http://localhost:8000 in your browser.
